# BPD Network Analysis Project

## Project Overview
This repository contains code for network analysis of BPD (Bronchopulmonary Dysplasia) microbiome data using SparCC and other methods.

## Directory Structure
- `network_analysis/keystone_analysis/` - Keystone taxa identification
- `network_analysis/sensitivity_analysis/` - Method comparison and sensitivity tests
- `scripts/` - Analysis pipeline scripts

## Analysis Pipeline
1. Network construction using SparCC
2. Sensitivity analysis with multiple correlation methods
3. Keystone taxa identification
4. Network visualization for Gephi

## Requirements
- R packages: dplyr, ggplot2, igraph, tidyr, reshape2
- Python with SparCC
- Bash shell
